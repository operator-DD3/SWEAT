SWEAT

A simple FORTH-like system created in Lua. Many features of an actual Forth system are yet to be implemented and any and all help in this project is greatly appreciated.

SWEAT currently supports:

  • Arithmetic Operations
  • IF-ELSE-THEN (buggy, must use ELSE & THEN)
  • Some common stack manipulations, such as DUP, DROP, SWAP, OVER, ROT, etc...

SWEAT currently DOES-NOT support:

  • DOES>
  • DO LOOP
  • VARIABLES (variables can be defined as : x 0 ; : x@ x @ ; : x! x ! ; )
